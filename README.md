## 🌟 项目简介

**人生计算器**是一个专业的在线生活规划工具平台，提供全面的时间管理、健康评估、财务规划、退休计算等功能。网站采用响应式设计，支持移动端和桌面端访问。

## ✨ 主要功能

### 📊 5个核心计算器
1. **时间与年龄计算器** - 精确到毫秒的时间计算，支持实时年龄更新
2. **健康与寿命评估计算器** - 基于健康习惯的寿命预测
3. **财务规划计算器** - 包含4个标签页的全面财务规划工具
4. **生活目标管理计算器** - 设定和跟踪人生目标
5. **退休规划计算器** - 详细的退休储蓄和支出规划

### 📚 6篇专业文章
- **时间管理**：高效利用碎片时间的实用技巧
- **财务规划**：个人财务规划和健康财务关系
- **健康生活**：延长寿命的科学习惯和健康生活方式
- **更多专题**：持续更新的专业文章库

### 🎨 特色功能
- ✅ **实时年龄计算** - 生日模式下的秒级实时更新
- ✅ **响应式设计** - 完美适配手机、平板和桌面

## 🔧 部署到GitHub Pages

### 自动部署
1. 将本仓库推送到GitHub
2. 进入仓库设置 → Pages
3. 选择`main`分支和`/ (root)`目录
4. 点击保存，几分钟后网站即可访问

### 自定义域名
1. 在`网站代码`目录添加`CNAME`文件
2. 在CNAME文件中写入您的域名
3. 在DNS提供商处配置CNAME记录
