## AdSense代码集成指南

### 1. 获取AdSense代码
1. 访问 https://www.google.com/adsense
2. 登录Google账户
3. 点击"获取代码"
4. 选择广告单元类型和尺寸
5. 获取广告代码

### 2. 代码插入位置
每个广告位占位符的替换方法：

```html
<!-- 替换前 -->
<div class="ad-placeholder">
    <i class="fas fa-ad"></i>
    <p>Google AdSense 广告位 - 728x90 横幅广告</p>
</div>

<!-- 替换后 -->
<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js"></script>
<ins class="adsbygoogle"
     style="display:inline-block;width:728px;height:90px"
     data-ad-client="ca-pub-XXXXXXXXXXXXXXXX"
     data-ad-slot="XXXXXXXXXX"></ins>
<script>
     (adsbygoogle = window.adsbygoogle || []).push({});
</script>
```

### 3. 广告位代码管理
建议为每个广告位创建独立的广告单元，便于管理和优化。

## 广告优化策略

### 1. 内容相关广告
- 确保广告与页面内容相关
- 使用关键词优化内容
- 分类明确的页面结构

### 2. 用户体验平衡
- 广告数量适中，不过度干扰
- 广告位置自然，不遮挡内容
- 移动端友好的广告尺寸

### 3. 广告格式选择
**优先推荐：**
1. 响应式广告（自动适配）
2. 固定尺寸广告（控制布局）
3. 匹配内容广告（内容相关）

### 4. 颜色和样式
- 与网站设计风格协调
- 适当的对比度
- 清晰的边框区分

## 合规性要求

### 禁止行为
1. ❌ 点击自己的广告
2. ❌ 鼓励他人点击广告
3. ❌ 将广告放置在无效点击区域
4. ❌ 隐藏广告
5. ❌ 自动刷新广告
6. ❌ 将广告放置在弹出窗口
7. ❌ 将广告放置在下载页面

### 必须遵守
1. ✅ 清晰的广告标识
2. ✅ 合理的广告密度
3. ✅ 内容相关的广告
4. ✅ 明确的隐私政策
5. ✅ 适当的年龄限制内容

## 性能监控

### 关键指标
1. **页面RPM**（每千次展示收入）
2. **点击率CTR**
3. **有效每千次展示费用eCPM**
4. **展示次数**
5. **点击次数**

### 优化建议
1. **测试不同广告位置**
   - A/B测试不同广告位效果
   - 调整广告密度
   - 优化广告尺寸

2. **内容优化**
   - 增加高价值关键词内容
   - 优化页面停留时间
   - 减少跳出率

3. **技术优化**
   - 提高页面加载速度
   - 优化移动端体验
   - 改善网站可用性

## 申请AdSense步骤

### 步骤1：准备网站
1. 确保所有页面完整
2. 添加足够的内容（建议30+篇文章）
3. 完善法律页面
4. 优化网站速度

### 步骤2：创建Google账户
1. 使用专业邮箱（非Gmail建议）
2. 确保账户信息真实
3. 验证手机号码

### 步骤3：提交申请
1. 访问 https://www.google.com/adsense
2. 点击"立即注册"
3. 填写网站信息
4. 提交审核

### 步骤4：等待审核
- 通常需要1-3个工作日
- 可能需要更长时间
- 保持网站内容更新

### 步骤5：安装代码
1. 审核通过后获取代码
2. 按本指南插入广告代码
3. 验证代码正常工作

## 常见问题解决

### 申请被拒原因
1. **内容不足**
   - 解决方案：增加更多原创内容

2. **导航问题**
   - 解决方案：优化网站结构

3. **版权问题**
   - 解决方案：确保所有内容原创

4. **技术问题**
   - 解决方案：修复网站错误

5. **政策违规**
   - 解决方案：阅读并遵守AdSense政策

### 收入优化技巧
1. **高价值关键词**
   - 财务、健康、投资类内容RPM较高
   - 创建相关专题内容

2. **用户互动**
   - 增加评论功能
   - 添加社交分享
   - 创建邮件列表

3. **流量来源**
   - SEO优化获取有机流量
   - 社交媒体推广
   - 内容营销
